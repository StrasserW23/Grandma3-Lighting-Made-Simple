Evolution of a Legend

Introduced in January 2018, the grandMA3 series from MA Lighting represents a radical redesign of lighting control, offering a new system architecture for increased data demands, built-in 3D visualization, and native GDTF support.

Launch & Hardware (2018)

Announced at Prolight + Sound 2017 and launched in 2018, the series replaced the industry-standard grandMA2. The hardware was reimagined for total control and longevity.

  • Interface: Dual encoders and multi-touchscreens for simultaneous attribute control.
  • Ergonomics: Motorized faders and a user-customizable interface focused on faster programming.

Work Without Boundaries

The grandMA3 system architecture was built to handle modern data networking far beyond traditional limits.

Connectivity:

LAN (Local): High-performance, real-time communication via MA-Net3.

WAN (Wide): Advanced capabilities for remote system management and cloud-based access over long distances.

System Expansion (2019-2020)

As the platform grew, MA Lighting introduced specialized hardware to fill the needs of different sectors.

  • 2019: Introduction of xPort Nodes DIN-Rail for large-scale data conversion.
  • 2020: Release of the onPC command wing XT, an all-in-one hardware solution for the onPC software.

Software Maturity

While early adoption faced a learning curve due to the entirely new operating system, it maintains core syntax familiarity with the grandMA2 while enabling more complex, faster programming workflows.
